Minimizing Distortion In Audio Steganography

PRESENTED AT THE 4TH IEEE CONFERENCE ON COMPUTING FOR SUSTAINABLE GLOBAL DEVELOPMENT, MAR 2017
Paper

Worked on cyber-security by removing distortion from the audio steganography technique like Parity coding, Least Significant Bit coding, Phase Coding. .This technique replaces each 4th layer LSB by data bit. This is way of masking data with a great bit rate of 16Kbps which is an optimum bit rate among all other techniques. By inducing the 4th layer LSB technique we tried to minimized the distortion in the stego file, inducing transparency. The masked data was recovered without any faults.
